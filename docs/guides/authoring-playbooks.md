# Authoring playbooks

Playbooks are literate-config Markdown — prose plus fenced, tagged, runnable blocks
(see the [schema](../specifications/playbook-schema.md)). You can let the model
write one from your live context (`create <prompt>`) or hand-author one in your
editor. This guide will cover writing front matter, tagging runnable steps, the
`{id=verify}` success check, and multi-language blocks.
