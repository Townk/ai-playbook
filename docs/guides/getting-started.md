# Getting started

ai-playbook is a terminal-native AI assistant that turns your live shell context
into runnable playbooks. Once installed, you invoke `assist` to get help with a
failing command or a question (it triages into a one-line command, a short answer,
or a full playbook), or `create` to author a playbook directly. This guide will
walk through installation, first run, and binding it to a shell key.
