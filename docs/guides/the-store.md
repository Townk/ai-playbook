# The store

The store is your accumulating library of playbooks — a global store under the data
dir and a project-local store committed inside a repository (see
[storage](../specifications/storage.md)). You browse and filter it with `list` and
`search`, view with `show`, and edit with `edit`; an external picker can consume the
`fuzzy-data-source` format. This guide will cover the two stores, slugs, and wiring
a fuzzy picker.
