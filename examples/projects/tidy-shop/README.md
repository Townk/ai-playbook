## tidy-shop

A small, complete, healthy fake project — a shell-script "shop" CLI with passing build and
test scripts. Used in chapters 01–02 to demonstrate the happy path: run, verify, rollback.

- `build.sh` — builds the project (always succeeds)
- `test.sh` — runs the test suite (always succeeds)
- `shop.sh list` / `shop.sh add <item>` — the tiny CLI
